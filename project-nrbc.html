NRBC HCI Project (VMware vSAN), Design & Deployment

Client

NRBC Bank PLC, Bangladesh

Problem Statement

Design and deploy a hyper-converged infrastructure using VMware vSAN to modernize the bank's data center, improve performance, and simplify management.

My Role

Lead System Engineer, responsible for the solution architecture, implementation, and performance tuning of the vSAN cluster.

The Solution

Implemented a highly available and scalable VMware vSAN cluster, providing a software-defined storage solution that integrates seamlessly with their existing VMware environment. This eliminated the need for a traditional SAN, reducing complexity and cost.

Technologies Used

VMware vSphere, VMware vSAN

Impact & Results

The vSAN solution delivered significant performance improvements for the bank's critical applications, simplified storage management, and provided a cost-effective and scalable platform for future growth.
