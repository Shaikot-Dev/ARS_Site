Nutanix HCI with Flow

Client

Jamuna Bank PLC, Bangladesh

Problem Statement

Implement a modern, secure, and scalable hyper-converged infrastructure using Nutanix, and leverage Nutanix Flow for micro-segmentation and network security.

My Role

Lead System Engineer, responsible for the solution design, deployment of the Nutanix cluster, and configuration of Flow security policies.

The Solution

Deployed a Nutanix HCI cluster to consolidate the bank's compute and storage resources. Implemented Nutanix Flow to create application-centric security policies, providing granular control over east-west traffic and protecting against advanced threats.

Technologies Used

Nutanix AHV, Nutanix Prism, Nutanix Flow

Impact & Results

The Nutanix solution provided a highly scalable and resilient platform for the bank's applications. Nutanix Flow significantly enhanced the bank's security posture by preventing the lateral movement of threats within the data center.
